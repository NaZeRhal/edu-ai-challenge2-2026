Review the built index.html and enhance all interactions.

## Checklist — verify each item works correctly

### Filters

- [ ] Year dropdown filters list correctly
- [ ] Quarter dropdown filters list correctly

- [ ] All three filters combine correctly

### Expand/collapse

- [ ] Clicking expand button on a row shows category stats
- [ ] Clicking again collapses it

- [ ] Expand animation is smooth

### Visual polish

- [ ] Rank 1 avatar has gold border
- [ ] Rank 2 avatar has silver border
- [ ] Rank 3 avatar has bronze border
- [ ] Rank badges correct colours (gold/silver/bronze)
- [ ] Podium height order correct (1st tallest, 2nd medium, 3rd shortest)
- [ ] Hover state on list rows
- [ ] Feedback button is fixed/floating at correct position

### Edge cases

- [ ] Search with no results shows empty state
- [ ] Filter combination with no results shows empty state
- [ ] Rank numbers re-number correctly after filtering
